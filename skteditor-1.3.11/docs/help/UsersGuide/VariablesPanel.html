Variables Panel

The Variables panel allows the user to view and edit the ISTP/IACG variables defined within the file.  Global attributes are used to provide information about the data set as an entity. An understanding of ISTP/IACG variables is required to use this panel.  Refer to the following sources for the definition and rules of use of ISTP/IACG variables:
  • Variables
  • ISTP/IACG Guidelines -- Variables (see https://spdf.gsfc.nasa.gov/istp_guide/variables.html)

Variable List

All variables defined in the current CDF are listed by name in the upper lefthand corner of the Variables panel.  Selection of a variable results in its definition being displayed on the rest of the Variables panel.

CDF Specifications

This section displays the CDF specification information for the selected variable.  The information displayed includes the following:
  • Name
  • Data Type
  • Time Varying
  • Dimensions
    • Format is  d:[s0, s1, ...sd-1]  where
      • d is the number of dimensions
      • sn is the number of elements in the nth dimension

Description

This section allows the user to view and edit the description attributes of the selected variable.  These attributes include the following:
  • Short Description
  • Long Description
  • Variable Notes
  • Keyword
    • class
    • subclass

Axis Information

This section allows the user to view and edit the axis information for the selected variable.  This information consists of the following:
  • Label 1
  • Label 2
  • Label 3
  • Scale Type
  • Format
  • Units

Plot Information

This section allows the user to view and edit the plot information for the selected variable.  This information consists of the following:
  • Variable Type
  • Display Type
  • Display Arguments

Depends

This section allows the user to view and edit the dependencies for the selected variable.

Virtual Variable

This section allows the user to view and edit virtual variable unique attributes of the selected variable.  This only appears if the selected variable is a virtual variable.  These attributes consist of the following:
  • Function
  • Components

Values

This section allows the user to view and edit the data values of the selected variable.  This only appears if the selected variable is ???support_data???.

Valid Min/Valid Max

This section allows the user to view and edit the valid minimum and valid maximum values for the selected variable.
