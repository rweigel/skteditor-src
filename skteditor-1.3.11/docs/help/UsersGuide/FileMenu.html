File Menu

The File menu contains the following items:
 
New Creates a new CDF file.  See ???tutorial??? for more information about creating new files
Open Opens an existing file in the editor.
Save Saves the file currently loaded in the editor using the file's current name.
Save As... Saves the file currently loaded in the editor with a name you provide using a file chooser dialog box.
Close Closes the file currently loaded in the editor.
Exit Quits the editor.
