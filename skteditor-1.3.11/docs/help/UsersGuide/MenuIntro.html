Menus

The menu bar at the top of the editor provides access to the operations that a user can perform in the editor.  The actual menu option that appear change depending upon the currently selected panel.  That is, one set of menus will appear when the Global Attributes panel is selected and a different set will appear if the Variables panel is selected.  The following menus are available:
  • File
  • Edit
  • Variable
  • Tools
  • Help
