Tools Menu

The Tools menu contains the following items:
 
ISTP Compliance Check Entire File Checks the entire file for ISTP guideline compliance.  A description of non-compliant issues are displayed in the messages window.
Global Attributes Checks the global variable definitions for ISTP guideline compliance.  A description of non-compliant issues are displayed in the messages window.
Attributes Editor   Opens an existing file in the editor.
Reset Fill Values   Saves the file currently loaded in the editor using the file's current name.
