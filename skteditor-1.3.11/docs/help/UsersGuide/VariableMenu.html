Variable Menu

The Variable menu contains the following items:
 
New Creates a new variable in the CDF file.
New Virtual Creates a new virtual variable in the CDF file.
Copy Creates a new variable by copying the CDF specification and attributes of the currently selected variable to a variable with a new name.
Delete Deletes the currently selected variable from the CDF file.
Rename Renames the current CDF variable.
Reset Fill Value
Check Checks the ISTP compliance of the currently selected variable.
Edit Specs
