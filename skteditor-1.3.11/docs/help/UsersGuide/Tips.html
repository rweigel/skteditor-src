Tips

Introduction

The following contains tips to aid the user with common issues.

Cut/Paste

You can copy and paste text from other applications into and out of this application using the Edit menu.  You may also use the accelerator keyboard shortcuts identified on the Edit menu.
 

Additional Attributes

It is common for individual projects to have global attributes and variable attributes that are not members of the Space Physics Guidelines for CDF (see https://spdf.gsfc.nasa.gov/sp_use_of_cdf.html) required or recommended set of attributes.  You may view and edit such attributes without leaving this application by using the Attributes Editor.  If the Attributes Editor is not sufficient or convenient, you may close (and save) the current file and then use any other tool to modify the file.  Once the modifications are complete, the file maybe reopened with this application.
