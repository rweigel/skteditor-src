The SKTEditor is a JavaTM application that greatly simplifies the creation, editing and correction of Common Data Format (CDF) files (see https://cdf.gsfc.nasa.gov/) that comply with the International Solar-Terrestrial Physics (ISTP) / Space Physics Data Facility (SPDF) guidelines (see https://spdf.gsfc.nasa.gov/sp_use_of_cdf.html). CDF files that are created using this tool will be able to be imported into systems like CDAWeb easily and with a minimum of human interaction.
This tutorial provides an overview of the SKTEditor features and execution.  Examples show key functionality  to manipulate the creation of ISTP compliant files, including:
  • Global Attributes Panel
  • Variable panel
  • Creation of new variables (non-virtual or virtual)
  • Compliance checking (variable or entire file)
  • Saving the final version of the file.
