This package contains standalone tools that are run against CDF files to gather information, check compliance against the ISTP standards, etc.

A wrapper in the bin directory, called .spdfJavaWrapper may be used to run any of these classes. If a new class is created, simply create a link to this file. Any arguments preceeded with a - will be passed to the java command, others will be passed to the application itself.

For an example, see CDFCheck
